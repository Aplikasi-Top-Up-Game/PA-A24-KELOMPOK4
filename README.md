# OUTPUT PROGRAM

## A.Tampilan Awal Program

### Pada tampilan awal program ini, akan ditampilkan pesan "Selamat Datang di Toko DDP Top-Up" diikuti dengan menu utama yang berisi opsi untuk login sebagai admin, login sebagai user, registrasi akun, dan keluar. Admin atau user kemudian diminta untuk memilih salah satu opsi yang tersedia.


![image](https://github.com/user-attachments/assets/30ff8145-4bcb-4dbd-8cef-1723d793bae7)


## 1. Registrasi Akun

### Pada tampilan awal saat user memilih opsi registrasi akun, user diminta untuk memasukkan username dan password baru. Tujuan registrasi ini adalah untuk menambahkan akun dengan password baru agar user dapat melakukan transaksi di toko DDP Top-Up. Fitur registrasi hanya tersedia bagi user saja.


![image](https://github.com/user-attachments/assets/a49c8266-80a5-4372-ac24-04d0a1e3bce3)


## B. Login User

### Pada tampilan awal saat user memilih login, program akan menampilkan menu login yang meminta user memasukkan username dan password yang sudah ada atau telah didaftarkan sebelumnya. Jika login berhasil, akan muncul pesan “SELAMAT DATANG {username}! ⸜(｡˃ ᵕ ˂ )⸝♡” (di mana {username} digantikan oleh username dari user yang berhasil login). Selain itu, akan ditampilkan menu user yang berisi opsi cek saldo, tambah saldo, transaksi top-up, cari game, sorting harga dan keluar. User kemudian diminta untuk memilih salah satu dari keempat opsi tersebut.


![image](https://github.com/user-attachments/assets/1ebe5e27-6b8a-447a-8e5a-4e369ec8f5e4)


## C. Menu User

### 1. Cek Saldo
### Jika user memilih opsi 1, program akan menampilkan informasi saldo yang dimiliki oleh user saat ini. Saldo ini merupakan jumlah yang tersedia dan dapat digunakan untuk berbagai transaksi dalam aplikasi.


![image](https://github.com/user-attachments/assets/6ed0b2a4-fab0-4b02-9779-54fd2410953f)


### 2. Tambah Saldo
### Jika user memilih opsi 2, program akan meminta user untuk memasukkan jumlah saldo yang ingin ditambahkan. Terdapat batasan jumlah minimal top-up sebesar 15.000 dan maksimal sebesar 2.000.000.


![image](https://github.com/user-attachments/assets/ad696fc1-7f1a-4196-83e0-a8f0b2990098)


### 3. Transaksi Topup 
### Jika user memilih opsi 3, program akan menampilkan daftar game yang tersedia untuk top-up, dan user akan diminta untuk memasukkan nomor game yang diinginkan untuk top-up. Sebagai contoh, jika user memilih game dengan nomor 2, yaitu Genshin Impact, maka program akan menampilkan pilihan paket top-up untuk game tersebut. User kemudian diminta untuk memilih paket yang diinginkan.


![image](https://github.com/user-attachments/assets/67adf405-2521-47c0-90e1-4e89a96436cb)
![image](https://github.com/user-attachments/assets/b844b267-f822-4b5d-b013-69ff148a9bbe)


### Misalnya, jika user memilih paket nomor 2, yaitu “60 crystal,” maka setelah pembelian berhasil, program akan menampilkan riwayat atau invoice dari top-up yang dilakukan. Setelah itu, user akan diarahkan kembali ke menu utama user.


![image](https://github.com/user-attachments/assets/3c11c1b4-3269-4213-93c8-e1c238187ffa)


### 4. Cari Game
### Saat user memilih opsi cari game, program akan meminta user untuk memasukkan nama game yang ingin dicari. Setelah itu, paket top-up yang tersedia untuk game tersebut akan langsung ditampilkan. Jika user memilih opsi 1, yaitu top-up game, mereka hanya perlu memasukkan nomor paket yang ingin dibeli. Apabila saldo mencukupi, transaksi akan diproses, dan invoice akan dibuat sebagai bukti pembelian.



![image](https://github.com/user-attachments/assets/4b5bc20c-ef67-4066-a200-33ce199b9e2a)
![image](https://github.com/user-attachments/assets/22b76fc7-5459-4330-b9e8-e0f6c1f75f88)



### 5. Sorting Harga
### Pada opsi sorting, user akan diminta untuk memasukkan nomor game yang ingin diurutkan. Setelah nomor game dimasukkan, program akan secara otomatis mengurutkan harga paket top-up untuk game tersebut dari yang paling murah hingga yang paling mahal



![image](https://github.com/user-attachments/assets/6ed9984b-0eee-4f6a-9839-2d75b1437871)


### 6. Keluar
### Ketika user memilih opsi keluar, program akan menampilkan pertanyaan, “Apakah Anda yakin ingin keluar dari menu? y/t.” Jika user memilih “t,” maka mereka akan diarahkan kembali ke menu user. Namun, jika user memilih “y,” program akan Kembali ke menu awal.


![image](https://github.com/user-attachments/assets/f618a130-8eb3-4c62-9386-56889c7a3406)
![image](https://github.com/user-attachments/assets/f2ff23a0-fe1f-46fc-8ba1-b3d8bbb0d2e9)


## D. Login Admin
### Ketika admin memilih opsi login, program akan meminta admin untuk memasukkan username dan password akun admin yang sudah ada di database. 


![image](https://github.com/user-attachments/assets/2b29956a-ed27-4400-9905-3967e12f8c48)


## E. Menu Admin
### Jika login berhasil, akan diucapkan “SELAMAT DATANG” sebagai sambutan, dan menu admin akan ditampilkan. Admin kemudian diminta untuk memilih salah satu dari lima opsi yang tersedia di menu tersebut.
